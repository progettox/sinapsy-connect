# Personas

## Brand Manager (Brand)
- Obiettivo: ottenere contenuti/servizi rapidi
- Pain: agenzie costose, creator non affidabili, pagamenti rischiosi
- Success: match veloce, consegna ok, review positiva

## Micro Creator (Creator)
- Obiettivo: monetizzare, trovare lavori coerenti
- Pain: ghosting, pagamenti in ritardo, truffe
- Success: escrow attivo, chat chiara, pagamento sbloccato

## Fotografo freelance (Service Provider)
- Obiettivo: ingaggi, portfolio visibile
- Pain: consegna file HD, contestazioni ingiuste
- Success: consegna tracciata + regole dispute
