# Flow - Project Workflow (MVP)

## Stati progetto
- `matched`
- `in_progress`
- `delivered`
- `completed`
- `disputed`

## Flusso
1. Match creato -> progetto `matched`
2. Creator inizia lavoro -> `in_progress`
3. Creator consegna file/link e preme "Lavoro concluso" -> `delivered`
4. Brand approva -> `completed`
5. Brand richiede revisione -> ritorno a `in_progress`
6. Se contestazione -> `disputed` fino a decisione supporto
