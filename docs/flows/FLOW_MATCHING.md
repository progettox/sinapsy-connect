# Flow - Matching

## Lato Brand
1. Dashboard
2. Create campaign (+)
3. Publish -> status `active`
4. Applications inbox
5. Accept -> status `matched` + create chat/workspace

## Lato Creator
1. Home swipe feed (campaign cards)
2. Swipe right -> create application (se requisiti ok)
3. My Applications -> puo abbandonare una richiesta `pending` e poi reinviarla
4. Notifica match -> entra in workspace
