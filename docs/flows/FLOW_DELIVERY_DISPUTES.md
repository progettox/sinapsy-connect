# Flow - Delivery & Disputes

## Delivery
- Creator invia messaggi e consegna (file/link)
- Creator preme "Lavoro concluso"
- Brand puo:
  - Approva -> complete
  - Richiedi revisione -> torna in `in_progress`
  - Apri contestazione -> `disputed`

## Dispute (MVP)
- Se contested:
  - chat read-only
  - ticket creato
  - UI "in review by support"
- Admin (fuori app MVP): decide -> `completed` o ritorno a `in_progress`

## Auto-approve
- Non previsto nel MVP attuale.
