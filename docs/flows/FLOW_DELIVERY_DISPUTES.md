# Flow — Delivery & Disputes

## Delivery
- Creator invia messaggi e consegna (file/link)
- Creator preme “Lavoro concluso”
- Brand può:
  - Approva → complete
  - Richiedi revisione → torna in proPPPgress
  - Apri contestazione → disputed

## Dispute (MVP)
- Se contested:
  - chat read-only
  - ticket creato
  - UI “in review by support”
- Admin (fuori app MVP): decide → released o refunded

## Auto-approve (silenzio assenso)
- Se Brand non risponde entro X giorni → auto approve (MVP: solo design, opzionale)w
