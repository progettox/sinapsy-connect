# Glossario

- **Collab / Campagna**: annuncio creato da un Brand.
- **Application / Candidatura**: richiesta del creator per una campagna.
- **Match**: il brand accetta una candidatura -> si crea chat.
- **Workspace / Project Room**: chat + consegna + stato progetto.
- **Dispute / Contestazione**: ticket aperto, blocca la chat in scrittura.
- **Watermark**: protezione asset (MVP puo essere solo placeholder UI).
