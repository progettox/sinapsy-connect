# Glossario

- **Collab / Campagna**: annuncio creato da un Brand.
- **Application / Candidatura**: richiesta del creator per una campagna.
- **Match**: il brand accetta una candidatura → si crea chat.
- **Workspace / Project Room**: chat + consegna + stato escrow.
- **Escrow**: fondi bloccati fino ad approvazione.
- **Dispute / Contestazione**: ticket aperto, blocca fondi e chat.
- **Take rate**: commissione della piattaforma.
- **Watermark**: protezione asset (MVP può essere solo placeholder UI).
