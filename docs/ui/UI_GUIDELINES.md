# UI Guidelines

## Style
- Premium, minimale, foto protagoniste
- Dark mode first
- accent color “electric” per CTA

## Components
- Card (campaign)
- Primary CTA button
- Secondary button
- Badge verified
- Stat chips (followers, engagement)

## Motion
- swipe fluido
- micro-animazioni leggere (match modal)

## Loading
- skeleton per card
- shimmer (opzionale)
