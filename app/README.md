# App Flutter

Cartella applicazione mobile Flutter.

## Run locale
1. `flutter pub get`
2. `flutter run`

## Analisi e build
1. `flutter analyze`
2. `flutter build apk --release`

APK release:
- `build/app/outputs/flutter-apk/app-release.apk`
